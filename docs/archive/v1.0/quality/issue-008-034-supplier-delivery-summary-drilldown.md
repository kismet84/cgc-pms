# ISSUE-008-034 供应商交期表现摘要与采购订单下钻收口报告

## 裁决

- 结论：通过。
- 阻塞：无。
- 范围：复用采购驾驶舱现有供应商交期表现摘要与采购订单页既有 `partnerId` 查询筛选，补齐最小下钻入口。
- 口径：本轮只完成采购订单交期履约摘要与下钻，不代表独立供应商履约档案、综合评级或完整供应商平台已完成。

## 实现结果

- 供应商摘要存在 `partnerId` 时，供应商名称使用现有 link button 调用路由 `push`，进入 `/purchase/order?partnerId=<id>`。
- 缺少 `partnerId` 时保持文本降级，不生成不可达链接；既有空态、订单数、逾期数、交期达成率与表现评分展示不变。
- `order.vue` 生产代码保持 no-op；复用既有路由参数恢复、查询同步与 `getOrderList` 调用链，恢复 `partnerId` 时保留其他 query。
- 采购订单路由继续受既有 `purchase:order:query` 权限守卫保护。
- 未新增后端接口、数据模型、migration、供应商评分口径或外部平台集成。
