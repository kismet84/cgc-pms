### 重大风险摘要

1. **[严重] 批量操作无记录数上限** — 所有批量保存接口（11处）无 @Size 限制，存在 OOM 风险
2. **[严重] 前端密码规则正则损坏** — 用户无法修改密码
3. **[严重] deploy/ 目录 .env 含生产级密钥明文存储** — JWT_SECRET、JASYPT_PASSWORD 等明文暴露
4. **[高危] CI 硬编码数据库密码** — `.github/workflows/ci.yml` 中明文密码
5. **[高危] 数据隔离遗漏** — checkContractBalance() 缺少 tenant_id 过滤
6. **[高危] 发票核验参数校验无效** — @NotEmpty Map 无法校验内容
7. **[高危] @Transactional 多处缺少 rollbackFor** — 3 处高危遗漏
8. **[高危] Docker Compose 端口未绑定 localhost** — MySQL/Redis/MinIO 端口暴露 0.0.0.0
9. **[高危] Flyway 迁移不可回滚 DROP COLUMN** — V54/V73 破坏性变更
10. **[高危] Spring Retry 完全缺失** — 手动重试不可靠
