## 5. 风险与发布边界

- 本报告只批准本地实现，不批准生产变更。生产必须按 `docs/database/database-remediation-deployment-runbook.md` 完成备份恢复、生产等价 preflight、人工审批和窗口观察。
- MySQL 旧迁移在空库执行时可能产生 `INSERT IGNORE`/兼容 DDL 警告；本次没有失败 migration，V195+ 不通过关闭外键检查掩盖孤儿。
- `overhead_allocation_record` 的生产外部消费者需要人工确认；未确认前保留表，不影响现行业务正确性。
- 工作区存在用户原有未提交改动；本任务未覆盖或清理无关改动，也未提交、push 或合并。
