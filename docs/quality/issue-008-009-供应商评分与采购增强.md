# ISSUE-008-009：供应商评分与采购增强 最小可行回归

结论：通过 / 非阻塞。

## 范围

- 复用采购经理驾驶舱现有采购订单查询结果，新增供应商评分最小聚合。
- 不新增 migration，不改生产凭据、生产数据库连接或生产发布配置。
- 不建设完整询价、报价、比价、定标或供应商黑名单平台。

## 实现

- 新增 `DashboardSupplierScoreVO`。
- `PurchaseManagerDashboardVO` 新增 `supplierScores`。
- 评分口径：按供应商统计当前订单范围内的订单数、逾期未完成订单数；`onTimeDeliveryRate = (订单数 - 逾期未完成订单数) / 订单数 * 100`，`performanceScore` 取交期达成率整数分。
- 前端同步 `DashboardSupplierScoreVO` 类型。
