## 权限、租户与数据边界

- 未登录详情返回401；已登录但无 `bid:query` 返回403。
- 持 `bid:query` 或管理员角色可读取当前租户记录。
- 跨租户 ID 与不存在 ID 使用相同 `BID_COST_NOT_FOUND`，响应不携带记录数据。
- 浏览器验收只读使用现有记录，没有创建、修改、删除、状态流转或测试数据重置。
