## 项目概述

CGC-PMS 是面向建筑工程总包项目的全过程管理系统，覆盖项目、合同、采购、库存、分包、成本、付款、结算、发票、审批、预警、驾驶舱、组织和系统管理等业务领域。

技术栈：Spring Boot 3.3 + Vue 3 + TypeScript + MySQL 8.0 + MyBatis-Plus + Flyway + Redis + MinIO + Docker。

## 关键架构决策

1. **模块化单体**：业务模块多、事务边界紧，选择模块化单体而非微服务。后续团队规模扩大后再考虑拆分。
2. **审批引擎自研**：从零构建审批引擎而非集成第三方，因为业务审批与成本生成、状态联动紧密耦合。
3. **MyBatis-Plus + Flyway**：放弃 JPA/Hibernate，选择 MyBatis-Plus 以保留 SQL 可控性；Flyway 管理所有 DDL/DML 变更。
4. **雪花 ID + JSON 字符串**：全局 `ASSIGN_ID` 生成 19 位 ID，JSON 序列化为字符串避免前端精度丢失。
5. **策略模式成本生成**：`CostGenerationStrategy` 接口解耦各业务模块的成本生成逻辑，新增来源只需实现策略即可。
6. **`isCritical` 事务一致性**：审批回调通过 `isCritical()` 控制异常是否回滚审批事务，保证审批状态与业务状态强一致。
