## 11. 独立复核与失败关闭

至少完成两类独立复核：

1. **安全复核：** Prometheus 网络边界、错误上报信任边界、敏感字段、Secret、租户/项目、生产凭据和文件隔离。
2. **测试复核：** 覆盖率质量、付款回归、MySQL/Flyway、两前端门禁、E2E、同 SHA CI 与目标环境证据完整性。

以下任一情况直接判定“不通过”并停止对应阶段：

- 指标端点可从公开网络匿名访问。
- 前端错误上报包含 token、Cookie、请求体、完整 URL 或高基数敏感标签。
- 付款重构改变金额、审批、事务、权限、租户或审计行为。
- MySQL TLS/迁移/数据守恒失败。
- P0 缺目标环境、专项授权、双人复核、对象守恒或回滚证据。
- CI 或目标证据未绑定同一待发布 SHA。
