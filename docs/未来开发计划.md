4. `SubMeasureServiceTest` 新增 7 个验证测试 ✅

### 2.2 ✅ 出库业务单据化（已完成 2026-06-24）

**方案**：新增「领料申请」模块（`MatRequisition`），Flyway V94 双环境迁移（`mat_requisition` + `mat_requisition_item` 双表）。`MaterialRequisitionWorkflowHandler.onApproved()` 自动调用 `MatStockService.stockOut("MAT_REQUISITION", ...)`。后端 8 文件（Entity/VO/Mapper/Service/Assembler/Controller/Handler），前端 7 文件（Type/API/Vue/Composables），路由 `/inventory/material-requisition`。

### 2.3 ✅ 项目自动归档通知（已完成 2026-06-24）

**方案**：`ProjectArchiveNotifier`（`@Async`）异步检测 4 条件（合同已结算+付款已付清+结算已定案+审批流完结），满足时发送站内通知。`SettlementWorkflowHandler.onApproved()` 末尾触发异步检查。设计边界：仅通知不自动归档。

### 2.4 ✅ 发票强制关联付款记录（已完成 2026-06-24）

**方案**：`InvoiceService.create()` 新增 `payRecordId` 非空校验 + `PayRecord` 引用完整性校验。前端 `InvoiceFormModal` 付款记录从"可选"改为"必选"。新增 RED 测试 12/13。
