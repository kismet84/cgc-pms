# v1.5 Iterations

本目录用于保存每轮 AutoPilot 的正式迭代记录。

v1.0 及更早记录见 [`docs/archive/v1.0/iterations/`](../archive/v1.0/iterations/)。

建议命名：

- `iteration-YYYY-MM-DD-plan.md`：本轮计划
- `iteration-YYYY-MM-DD-report.md`：本轮结果

每份 report 至少记录：

- Issue 编号与标题
- 修改文件
- 验证命令与结果
- 是否自动合并
- 是否进入下一轮
