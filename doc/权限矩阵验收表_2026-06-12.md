## 已知问题

| ID | 问题 | 影响 | 建议 |
|----|------|------|------|
| P-403 | AuthorizationDeniedException 返回 500 而非 403 | 前端无法区分"未授权"和"服务器错误" | GlobalExceptionHandler 中增加 AuthorizationDeniedException → 403 映射 |

---

## 结论

权限矩阵在非 admin 角色上生效正确：
- 系统管理 API 仅 Admin 可访问
- 业务管理 API 仅 Admin 可访问（当前未实现细粒度角色权限，所有业务 API 由 ADMIN 或 SUPER_ADMIN 角色控制）
- 审批工作流 API 对所有已认证用户开放（由工作流引擎进行任务级别权限控制）
- AuthorizationDeniedException 返回码问题不影响功能正确性
