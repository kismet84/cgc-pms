  - 新增：`database/migration/V32__add_submit_permissions.sql`

- 新增迁移完整性回归测试：
  - `backend/src/test/java/com/cgcpms/MigrationIntegrityTest.java`

### 2. 工作流租户隔离与加签修复

- `WorkflowEngine`：
  - 审批/驳回幂等键写入使用任务租户，不再固定写入 `tenant_id = 0`。
  - 审批记录写入优先使用调用方已持有的实例/任务上下文，避免兜底查询失败后回落到租户 0。
  - 加签前校验任务必须为 `PENDING`，并校验实例运行中、节点为活动节点。
- `WorkflowQueryService` / `WorkflowController`：
  - 我的待办、实例详情、可用操作增加当前租户过滤。
- `MyMetaObjectHandler`：
  - 插入自动填充租户时不再覆盖实体显式设置的 `tenantId`。
- 新增回归用例：
  - 工作流记录和幂等键保留实例租户。
  - 已处理任务禁止加签。

### 3. 成本偏差展示口径修复

- `frontend-admin/src/pages/cost/summary.vue`
  - 将 `costDeviation` 从百分比展示改为金额展示，单位为万元。
  - 文案从“成本偏差率/偏差率”调整为“成本偏差/成本偏差(万元)”。
- `frontend-admin/src/pages/dashboard/index.vue`
  - 同步仪表盘 KPI、钻取摘要、表格列的金额展示口径。

### 4. H2 测试环境一致性修复

- `db/h2/data.sql`：
  - 固定主键 seed 数据从 `INSERT` 改为 `MERGE INTO ... KEY(id)`，避免多测试上下文复用 H2 内存库时重复主键。
  - 补充材料验收审批模板和节点 seed。
- `db/h2/schema-phase23.sql`：
  - 补齐 `ct_contract_payment_term.payment_condition`、`actual_date`、`term_status`。
- `RegressionFixVerificationTest`：
  - Raw SQL 不再依赖 MySQL `DATE_ADD(... INTERVAL ...)`，改为 Java 参数传入日期，兼容 H2。
