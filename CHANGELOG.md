# 更新日志

## 1.0.0 - 2026-07-11

### 核心能力

- 建立项目立项、合同与签证、采购与库存、成本归集、付款发票、结算、审批预警和经营驾驶舱等工程总包项目管理能力。
- 提供项目经理、商务经理、采购经理和生产经理的受控驾驶舱；总工程师驾驶舱继续保持冻结，等待真实技术域数据契约。
- 建立鉴权、租户与项目边界、审计留痕、数据库迁移以及前后端测试和 CI 门禁。
- 建立本地 AutoPilot 串行无人值守控制面，包括严格 Ready 合同、隔离 worktree、证据绑定、独立审查、有限修复和安全恢复。

### 发布边界

- 当前版本作为 1.0 稳定基线归档；生产部署仍需按项目上线门禁独立审批。
- AutoPilot 保持 `autoPush=false`、`maxParallel=1`，不自动发布生产、不连接生产数据库。
- 并发 AutoPilot、总工程师驾驶舱和新增业务域不属于 1.0 范围。
