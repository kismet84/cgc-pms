# Agent Message Records

This directory stores short, document-linked message records for local multi-agent collaboration.

Messages should contain only:

- a link/path to the detailed task, report, or notice document;
- one short sentence describing the message.

Full details belong in:

- `.agent-runtime/tasks/`
- `.agent-runtime/reports/`
- `docs/agents/`

Do not put long implementation instructions, logs, stack traces, or complete test output in message files.
