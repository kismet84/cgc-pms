=== Key Design Decisions ===
- Optimistic lock via @Version (MyBatis-Plus auto-detection)
- Stock never negative (business rule in application layer, not DB constraint)
- Thread-safe concurrent operations via version-based retry (max 3)
- DuplicateKeyException catch for concurrent first-insert scenario
- tenantId from UserContext + LambdaQueryWrapper.eq(tenantId) on every query
- @Transactional on stockIn/stockOut for atomicity
- Txn journal records available_after snapshot
